This is a TicTacToe in CMD(.bat)!

You have two gamemodes in this game, PvP - Player VS Player, PvE - Player vs Engine(AI)
In PvE you can choose difficulty from easy to hard.
This game also keeps count of how many wins each player has(in a single game session).

The final version is 'TicTacToe.bat'
The older version is 'TicTacToe_beta.bat'

Both were tested on Windows 10 Pro edition.
